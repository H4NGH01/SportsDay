# SportsDay
Macau SMP SportsDay papermc plugin
## Commands
* `/competition`
> Competition hosting command
> * `start <competition_id>`
> * `end`
> * `join <player> <number>`
> * `leave | leave <player>`
> * `score <player> <new_score>`
> * `info`
* `/competitiongui`
> Open competition GUI
> * `book`
